> ## 2017년도 마케팅 조사
> 공유경제서비스 이용의도 검증 - 카카오서비스 이용자 설문 및 가설검증

목적: 국내 연구기관의 보고서를 포함한 자료를 이용하여 공유경제, 카카오 플랫폼, 이용 용이성, 사회적 자본, 이용의도의 개념과 정의를 알아보고 이에 관련된 참고문헌과 빅데이터를 활용하여 연구 모형을 설계하고 연구 가설을 도출 및 검증
 - 설문조사를 통한 데이터 수집
 - SPSS를 활용한 통계적 분석
